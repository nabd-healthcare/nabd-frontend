# Nabd Landing Page

## 📋 Overview
This is a complete, modern landing page for the Nabd medical platform. The page is built with React and Tailwind CSS, following the brand colors and design guidelines.

## 🎨 Brand Colors
- **Primary (Teal Medical)**: `#1C8B8F`
- **Dark Variant (Deep Teal)**: `#14666A`
- **Secondary (Soft Navy)**: `#1F2E3C`
- **Neutral (Cool Gray)**: `#E7ECEF`
- **Background (Snow White)**: `#FFFFFF`

## 📁 Structure
```
src/pages/Landing/
├── LandingPage.jsx          # Main landing page component
├── index.js                 # Export file
└── components/
    ├── Header.jsx           # Navigation header with mobile menu
    ├── Hero.jsx             # Hero section with AI card demo
    ├── Features.jsx         # 6 feature cards
    ├── AISection.jsx        # AI capabilities showcase
    ├── HowItWorks.jsx       # 4-step process
    ├── OurDoctors.jsx       # Featured doctors section
    ├── Testimonials.jsx     # User reviews
    ├── BlogSection.jsx      # Latest medical articles
    ├── FAQ.jsx              # Collapsible FAQ section
    ├── DownloadApp.jsx      # Mobile app download CTA
    ├── CTA.jsx              # Call-to-action section
    └── Footer.jsx           # Footer with links and social
```

## 🚀 Sections Included
1. **Header** - Sticky navigation with dual CTA buttons
2. **Hero** - Main section with AI diagnosis card visual
3. **Features** - 6 key features with icons and hover effects
4. **AI Section** - Dark section showcasing AI capabilities
5. **How It Works** - 4-step user journey
6. **Our Doctors** - Featured doctors with ratings
7. **Testimonials** - User reviews from patients and doctors
8. **Blog Section** - Latest medical articles
9. **FAQ** - Frequently asked questions
10. **Download App** - Mobile app promotion
11. **CTA** - Final call-to-action
12. **Footer** - Complete footer with links

## 🔗 Routing
The landing page is accessible at:
- `/` - Root path (main landing page)
- `/landing` - Alternative route (same page)
